# Opus Clone

A pixel-accurate clone of [Opus – Framer template](https://opus-template.framer.website) built with React, Vite, TypeScript, Tailwind CSS, and Framer Motion.

## Tech stack

- **React 19** + **Vite 7**
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** (animations)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Assets

- **Project cards** (Projects): placeholder blocks; replace with original images/videos.
- **Blog cards** (Blog): placeholder blocks; replace with original thumbnails.
- **Hero**: optional background image/video can be added; see `Hero.tsx` comments.

Replace placeholders with assets from the original site or your own media.
